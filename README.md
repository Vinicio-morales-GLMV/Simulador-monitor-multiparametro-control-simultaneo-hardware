VitalSign Sim: Monitor Multiparámetros con Gemelo Digital
Este proyecto es un sistema híbrido de monitoreo médico desarrollado para la carrera de Ingeniería Biomédica. Consiste en un hardware basado en Arduino (simulado en Wokwi) que procesa señales vitales reales/simuladas y un dashboard web dinámico que actúa como el monitor de cabecera del paciente.

Características Principales
Procesamiento en Tiempo Real: Lectura de 4 parámetros vitales mediante potenciómetros (simulando sensores médicos).

Visualización Dual: 
Hardware: Pantalla LCD 16x2 para datos numéricos y OLED para la curva ECG.

Web: Interfaz dinámica desarrollada con HTML/JS que refleja el estado del paciente.

Algoritmo de Triaje: Sistema de alertas automáticas (Leds y Buzzer) basado en rangos clínicos.

Generador de Curva ECG: Simulación matemática del complejo QRS sincronizado con los BPM (latidos por minuto).
