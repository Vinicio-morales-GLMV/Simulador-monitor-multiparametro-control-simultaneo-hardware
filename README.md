# VitalSign Sim - Monitor Multiparámetros con Gemelo Digital

## Descripción
Sistema híbrido de monitoreo médico desarrollado para Ingeniería Biomédica.
Combina hardware basado en Arduino (simulado en Wokwi) con un dashboard 
web dinámico que actúa como monitor de cabecera del paciente.

## Tecnologías utilizadas
- Arduino (simulado en Wokwi)
- HTML / JavaScript
- Python

## Características principales
- Lectura de 4 parámetros vitales mediante potenciómetros
- Pantalla LCD 16x2 para datos numéricos
- Pantalla OLED para curva ECG
- Algoritmo de triaje con alertas automáticas (LEDs y Buzzer)
- Simulación matemática del complejo QRS sincronizado con BPM

## Funcionalidades
- Visualización en tiempo real de signos vitales
- Interfaz web dinámica como monitor de cabecera
- Sistema de alertas basado en rangos clínicos
